# customer-churn-prediction
# Прогнозирование оттока клиентов компании

Разработка системы прогнозирования оттока клиентов с использованием методов машинного обучения. В проекте сравниваются три алгоритма классификации, лучший из которых определяется по метрике ROC-AUC.

---

## Описание

Задача — предсказать, уйдёт ли клиент из компании, на основе данных о его поведении и характеристиках.

В ходе работы:
- проведён анализ и визуализация данных (распределения, boxplot, столбчатые диаграммы, матрица корреляций)
- выполнена предобработка данных (устранение пропусков, кодирование категориальных признаков)
- обучены и сравнены три модели: логистическая регрессия, случайный лес, CatBoost
- подобраны гиперпараметры с помощью RandomizedSearchCV
- выбрана лучшая модель по метрике ROC-AUC

## Датасет

Датасет содержит две выборки: обучающую (`train.csv`) и тестовую (`test.csv`).

Основные поля:

| Поле | Описание |
|------|----------|
| `ClientPeriod` | Длительность пользования сервисом (дни) |
| `MonthlySpending` | Ежемесячные траты |
| `TotalSpent` | Общие траты за всё время |
| `Sex` | Пол клиента |
| `HasPartner` | Наличие партнёра |
| `HasChild` | Наличие детей |
| `HasInternetService` | Наличие интернет-услуги |
| `PaymentMethod` | Способ оплаты |
| `Churn` | Целевой признак: ушёл клиент (1) или нет (0) |

## Запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/ваш-username/customer-churn-prediction.git
cd customer-churn-prediction
```

2. Поместите файлы `train.csv` и `test.csv` в корень проекта.

3. Откройте `customer_churn.ipynb` в Jupyter Notebook или Google Colab.
